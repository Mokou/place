
# How to Add an Overlay

1. Create a 1:1 pixelart
2. Run [dotter.py](https://github.com/Mokou/place/tree/main/scripts) on it.
3. Upload it to the [overlays](https://github.com/Mokou/place/tree/main/overlays) folder.
4. Update [overlays.txt](https://github.com/Mokou/place/tree/main/overlays/overlays.txt) with the required info. The numbers before the filename are the x and y coordinates of the top left pixel.
5. Done.
